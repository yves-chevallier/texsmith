# Progress Bars

The `texsmith.progressbar` extension renders Markdown shorthand into LaTeX progress bars:

```
[=25% "Research"]
[=50% "Implementation"]
[=75% "Review"]{: .candystripe}
[=100% "Launch"]{: .thin}
```
