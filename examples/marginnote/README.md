# Margin Notes

Showcases the `{margin}[窶ｦ]{side?}` inline syntax provided by
`texsmith.extensions.marginnote`. Notes are rendered with LaTeX's
`marginnote` package (auto-loaded by the `ts-extra` fragment on
detection of `\marginnote`).

Build with:

```sh
make all
```

The resulting PDF lives under `build/tectonic/marginnote.pdf`.
