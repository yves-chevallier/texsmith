# Index Demonstration

Welcome to the `texsmith-index` demo. Tags authored with the syntax `#[...]`
are tracked globally and exposed to both TeXSmith and MkDocs.

## Key Concepts

- The #[TeXSmith]{i} engine converts Markdown into LaTeX.
- Its #[Rendering]{b} pipeline cooperates with the index extension.
- Use #[Matrices][Determinant][Formula]{bi} to showcase three-level hierarchies.

## Exercises

Revisit the #[Matrices][Determinant] definition to verify that the primary tag
is reused across sections.
