TeXSmith Logos Extension

This small demo page illustrates the HTML output produced by the texsmith-texlogos extension. Each keyword below is produced by simply typing the plain text variant in a Markdown paragraph:

Command Markdown Input Rendered HTML
\TeX{} TeX TEX
\LaTeX{} LaTeX LATEX
\LaTeXe{} LaTeX2e LATEX2ε
\TeX{}\textsc{Smith} TeXSmith TeXSmith

Tip: Hover the rendered logos to reveal their description tooltips.

TeX, LaTeX, LaTeX2e, and TeXSmith are all rendered with their
respective logo macros.
