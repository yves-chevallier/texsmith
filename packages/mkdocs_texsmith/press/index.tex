\section{MkDocs LaTeX Plugin}\label{mkdocs-latex-plugin}

This Plugin uses behind-the-scenes TeXSmith to convert your MkDocs documentation into a beautiful PDF document.

\subsection{Features}\label{features}

\begin{itemize}
\item{} Convert your entire MkDocs site into a single PDF file.
\item{} Support for custom LaTeX templates.
\item{} Automatic handling of images and links.
\item{} Configurable options for page size, margins, and fonts.

\end{itemize}
\subsection{Installation}\label{installation}

To install the MkDocs LaTeX Plugin, use pip:

%\leavevmode % Required to avoid a bug in the tcolorbox package
\begin{code}{text}{}{}
pip install mkdocs-latex
\end{code}
