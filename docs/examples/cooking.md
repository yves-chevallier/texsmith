# Cooking Recipes

TeXSmith can be used to write cooking recipes with professional formatting from
a YAML description and a custom template.

=== "cake.yml"

    ```yaml
    ---8<--- "examples/recipe/cake.yaml"
    ```

=== "template.tex"

    ```tex
    ---8<--- "examples/recipe/template.tex"
    ```
